### Control Limits Calculator


#### Basic Functionality

- Accept a CSV with mixed QC data
- Sort the different QC types
- Return calculated control limits for each QC Type
- Export formatted results to a file


#### Future Features


- Accept some inputs via UI/Template:
    - Method ID
    - LOQ
    - Input Filename
    - Output Filename
- Interactive outlier flagging and omission
- Comprehensive reporting
    - Include dataset
    - Flag omissions with user provided comments
    - Output format of choice
